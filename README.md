<h1 align="center">📊 Government Complaint & Performance Dashboard</h1>

<p align="center">
  A modern web application to <b>raise complaints against government ministries</b> and analyze their <b>performance, budget allocation, and project progress</b>.
</p>

<p align="center">
  🚨 Raise complaints • 📈 Track complaint status • 💰 Budget transparency • 🏛️ Ministry performance insights
</p>

---

## ✨ Key Features

* 📝 **Raise Complaints** against different government ministries
* 📊 **Track Complaint Status** in real-time
* 🏛️ **Ministry Performance Dashboard** with analytics
* 💰 **Budget Transparency** and allocation insights
* 📁 **Project Tracking System** for ministry initiatives
